# Loading

Renders a Loading component. Comes in different sizes

## Sizes

xs, sm, md, lg

<Canvas>
  <Loading size="xs"/>
  <Loading size="sm"/>
  <Loading size="md"/>
  <Loading size="lg"/>
</Canvas>

## Colors

primary, secondary, tertiary, white, black, success

<Canvas>
  <Loading color="primary"/>
  <Loading color="secondary"/>
  <Loading color="tertiary"/>
  <Loading color="white"/>
  <Loading color="black"/>
  <Loading color="success"/>
</Canvas>

It is possible to also use "currentColor". Example of Loading component inside of `<Text color="danger">`

<Canvas>
  <Text color="danger" size="lg">
    <Loading color="currentColor"/>
    <Text i>Loading...</Text>
  </Text>
</Canvas>
