# Button

Button component works as Flex item, centered by default.

**Adding icon:**

If you want to add icons as prefix/suffix, you can do it by adding them as children.
If you do it, it is recommended to add Text component as well, because without it, everything is going to be rendered in `p` tag, which ruins centering of the icon.
By default, icons (svg elements) inside button have font-size of 1.25em, but you can change with "size" property on icon.

```
<Button type="dangerOutline">
  <Text>Button with slightly bigger icon</Text>
  <RxTrash size="1.5em" />
</Button>
```

## Special props

- disabled - changes opacity to 0.7 and pointer-events to none (default: false)
  <Button disabled>Button disabled</Button>
- fullWidth - sets width to 100%, otherwise it is "fit-content" (default: false)
  <Button fullWidth>Button fullWidth</Button>
- icon - makes button as square. This type of button expects only icon as children (default: false)
  <Button type="dangerFill" icon>
    <RxTrash />
  </Button>
