# Avatar

Renders an avatar component.

## Props

```typescript
type AvatarSize = 'xs' | 'sm' | 'md' | 'lg' | 'xl' | '2xl';
type AvatarType = 'unknown' | 'user' | 'character' | 'world' | 'ruleset';

interface AvatarProps {
  url?: string;
  type?: AvatarType; //default is 'unknown'
  size?: AvatarSize; //default is 'lg'
  fallbackText?: string;
}
```

## Default

By default, size is "lg"

<Canvas>
  <Story of={stories.storyDefault}/>
</Canvas>

## No url

It is possible to render default avatar, without giving out URL. If that is the case, it is possible to pass in a type, which will be used to display correct "unknown" image.

Example of "unknown" user avatar:

<Canvas>
  <Story of={stories.storyNoUrl}/>
</Canvas>

## Sizes

It is possible to render different sizes of avatar. By default, size is "lg".

```typescript
xs = 16; //using 30x30 variant
sm = 24; //using 30x30 variant
md = 36; //using 100x100 variant
lg = 48; //using 100x100 variant
xl = 100; //using 100x100 variant
2xl = 200; //using 200x200 variant
```
